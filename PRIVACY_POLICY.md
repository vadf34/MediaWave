# Privacy Policy for MediaWave

This Privacy Policy governs the use of the mobile application **MediaWave** (hereinafter referred to as the "Application") for Android devices, created by **nexid** (hereinafter referred to as the "Developer") as an Open Source app.

### 1. Information Collection and Use
The Application is designed to download media files using the open-source yt-dlp engine. 
- **No Personal Data Collection:** The Developer does not collect, store, share, or transmit any personally identifiable information, account credentials, or user data.
- **No Tracking/Analytics:** The Application contains no third-party analytics tools, tracking SDKs, or advertisements.

### 2. Device Permissions Required
To function properly, the Application requires the following permissions on your Android device:
- **Internet Access:** Required to connect to media hosting platforms, parse video/audio links, and download files. It is also used to download and update the yt-dlp binary upon the first launch.
- **Storage/Media Access:** Required to save the downloaded video files into `Movies/MediaWave` and audio files into `Music/MediaWave`.
- **Clipboard Access:** Used solely to automatically detect and paste media links when you open the Application. No clipboard data is ever sent to external servers.
- **Foreground Service:** Required to ensure that your active media downloads are not interrupted when the Application is minimized or the screen is turned off.

### 3. Cookies Management
The Application allows users to import cookie files from their web browsers to enable downloading content from restricted or private accounts. These cookies are stored strictly locally on the user's device within the secure Application sandbox and are never transmitted to the Developer or any third parties.

### 4. Third-Party Services
The Application utilizes the third-party open-source library `youtubedl-android` (a wrapper for `yt-dlp`). When downloading media, the Application interacts directly with the respective third-party media hosting platforms (e.g., YouTube, TikTok, Instagram, etc.). These platforms may collect data in accordance with their own privacy policies.

### 5. Changes to This Privacy Policy
The Developer may update this Privacy Policy from time to time. You are advised to review this page periodically for any changes.

### 6. Contact Us
If you have any questions regarding privacy while using the Application, please open an issue on the official GitHub repository: https://github.com/vadf34/MediaWave
